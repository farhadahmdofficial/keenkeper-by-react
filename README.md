# KeenKeeper - Personal Relationship Manager 🤝

**KeenKeeper** is a sleek, minimalist web application built to help users nurture and maintain their personal connections. In an age of social media noise, KeenKeeper focuses on what matters: tracking meaningful interactions, setting relationship goals, and visualizing your social health through data analytics.

The project is a showcase of high-quality UI/UX implementation, turning a Figma vision into a fully functional React application.

## 🚀 [Live Demo](https://keen-keper-by-farhad-reactrouter.netlify.app/)

## ✨ Features

- **Dynamic Interaction Timeline:** A central hub to view every call, text, and video chat in chronological order. Includes smart filtering by activity type.
- **Friendship Analytics:** Visual representation of communication habits using **Recharts**, helping users identify who they need to catch up with.
- **Quick Check-In System:** One-click logging for interactions (Call, Text, Video) with real-time global state updates.
- **Relationship Goal Tracking:** Set specific goals (e.g., "Contact every 10 days") and stay accountable.
- **Responsive Design:** Mobile-first approach using **Tailwind CSS** and **DaisyUI** for a seamless experience across all devices.
- **Real-time Notifications:** Integrated **React Hot Toast** for instant feedback on user actions.

## 🛠️ Tech Stack

- **Framework:** React.js (v18+)
- **Styling:** Tailwind CSS, DaisyUI
- **Routing:** React Router 6
- **State Management:** React Context API
- **Data Visualization:** Recharts
- **Feedback:** React Hot Toast
- **Deployment:** Netlify / Vercel
